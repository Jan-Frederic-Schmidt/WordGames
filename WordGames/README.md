# HOW TO USE:

The usage should be obvious from the app itself. There is currently one mode to play in: *Wordle*. In *Wordle* mode you have to guess a random five-letter word by writing other five-letter words to find out which letter is where. But I'm sure you're well aware what *Wordle* is. If not, look [here](https://www.nytimes.com/games/wordle/index.html)
